Video Downloader

(A browser-based wrapper for yt-dlp.)

Click for Help
  • Radio buttons: is this a request for a Youtube video, or for a Tiktok video? Default: Youtube video
    • Youtube video: From the YouTube URL after the video, enter the string after "v=".
    • Tiktok video: Enter the whole of the Tiktok URL up to any question mark, if present. Exclude the question mark and anything to its right. For example: https://www.tiktok.com/@polkamot/video/705579994389055237
  • "sub-folder if required" - optional
      The output video(s) is saved under the folder [data] in the app location on the server.
      Enter a value here if you want a sub-folder under [data].
      For Example, "JumpRope" will create a folder [data/JumpRope].
      Spaces: If the sub-folder you request has spaces, each word is capitalized, and the spaces removed.
      For example, "jump rope" would become "JumpRope".
  • Download: the video is saved to the data folder or a sub-folder of data. See above.
